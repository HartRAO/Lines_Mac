1720JY [memory]

convert 1720 MHz CH3OH spectrum from Kelvin to Jansky, 
using built-in table of Point Source Sensitivity as a function of date

This routine automatically applies the appropriate point source
sensitivity for the period of the observation to spectra obtained at
1710  - 1730 MHz.

The calibration is for LCP spectra, scaled as though the source is
unpolarised, to represent the total intensity.

Note that at 1720 MHz, PSS(RCP)/PSS(LCP) = 0.992 +- 0.003
This is assumed to be constant over the period.

If the memory is not specified the default set memory is used.

known spectra sensitivity changes from HartRAO dish surface upgrade
derived from interpolating 3.5 and 6cm continuum calibrators
