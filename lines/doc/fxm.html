FXM [memory]

Median filter X values with median filter 2 = 4253H,Twice

For spike removal and smoothing of X values (ordinates).

This is useful for example for drift scans where jumps in the angle may
occur, making working with the data difficult.

Taken from "Applications, Basics and Computing of Exploratory Data Analysis"
by Paul F Velleman and David C Hoaglin, Duxbury Press, Boston, 1981.
