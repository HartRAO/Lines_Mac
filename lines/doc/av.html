AV [from memory] [with memory]

Combine spectrum from specified memory with spectrum in second memory.

This is used to add spectra into an average spectrum.  The spectra are
added weighted according to their integration time.  The spectrum to add
is in the "from" memory, and the average spectrum is in the "with"
memory.

If the memories are not specified the user is prompted for them.

example:
   
AV 1 2
